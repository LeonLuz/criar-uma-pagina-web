Entendendo Comunicação Client X Server

O primeiro módulo do curso abrange a História da Web, explicando o contexto geopolítico onde surgem os primeiros computadores e os grandes nomes que contribuiram para a criação do que hoje conhecemos como a Internet. Em seguida, o professor descreve o que são os Clients e o que são os Servers e exemplifica como hospedar um site.

  • Clients: representados pelos computadores, tablets e celulares, são os dispositivos que consomem informações da Internet, e se comunicam através da Internet com outros computadores.
  • Servers: podem ser locais ou conectados na Web e servem para armazenar arquivos e e-mails, fazer o streaming de dados, monitorar a segurança e gerenciar arquivos html, css e JavaScript.

Por fim, são apresentadas as diferentes linguagens de programação e de que maneira são utilizadas.

Introdução ao HTML na Prática

O segundo módulo do curso aborda a estrutura básica do HTML utilizando o VS Code para exemplificar a construção de uma página HTML. Foram apresentadas as tags e seus atributos básicos. Uma página html deve se iniciar com uma tag "html" de abertura e uma de fechamento, seguida pela tag "title" que nomeia a aba da página para então abrir a tag "body", esta contém todo o contéudo que aparece para o usuário.

Desafio de Projeto

A fim de exercitar o contéudo abordado, a DIO propôs este desafio de projeto que visa a construção de uma página web que utilize todas as tags apresentadas em aula para compor um repositório no GitHub.
